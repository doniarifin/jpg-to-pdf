# JPG & PNG to PDF Converter

A simple and fast web application to convert JPG and PNG images into a single PDF file.  
Built with React, TypeScript, and Vite, with a focus on clean UI and smooth user experience.

---

## Features

- Convert multiple JPG and PNG images into one PDF
- Drag and drop to reorder pages
- Real-time PDF preview with page layout
- Adjustable settings:
  - Page orientation (portrait / landscape)
  - Page size (A4, Letter, Legal)
  - Custom margins
- Delete individual pages before generating PDF
- Automatic file naming based on the first uploaded image

---

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- jsPDF
- dnd-kit

---

## Getting Started

### Install dependencies

```
npm install
```

### Run development server

```
npm run dev
```

### Notes

- PNG images are supported, including transparency
- Output file name is automatically generated from the first uploaded file
- All processing happens in the browser (no backend required)
